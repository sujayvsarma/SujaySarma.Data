# SujaySarma.data

This repository contains the codebase for the following Nuget packages:

NuGet Package | Code path 
--------------|-----------
SujaySarma.Data.Core | [here](https://github.com/sujayvsarma/SujaySarma.Data/tree/master/SujaySarma.Data.Core)
SujaySarma.Data.Azure.Tables | [here](https://github.com/sujayvsarma/SujaySarma.Data/tree/master/SujaySarma.Data.Azure.Tables)
SujaySarma.Data.Files.TokenLimitedFiles | [here](https://github.com/sujayvsarma/SujaySarma.Data/tree/master/SujaySarma.Data.Files.TokenLimitedFiles)
SujaySarma.Data.SqlServer | [here](https://github.com/sujayvsarma/SujaySarma.Data/tree/master/SujaySarma.Data.SqlServer)
SujaySarma.Data.WindowsRegistry | [here](https://github.com/sujayvsarma/SujaySarma.Data/tree/master/SujaySarma.Data.WindowsRegistry)

There are also some basic-level unit tests in the `SujaySarma.Data.Tests` but these currently cover only the functionality of `Core` and nothing else. 

The previous locations of these libraries (pre v9.0.0) have been made read-only and archived. All further development for all of these libraries shall proceed in this repo until a future change.
